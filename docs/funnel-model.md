# 퍼널 모델: 3단계 압축

## 현황 (2026-03 기준)
- 인스타 팔로워: 6,000명
- 월 DM 문의: 3~5건
- 카카오톡 전환율: 30~50% (추정)
- 계약 전환율: 20~30% (추정)

## 퍼널 3단계

### 병목1: 콘텐츠 → DM 유입
- 현재: 월 3~5건
- 목표: 월 18~22건
- 핵심 레버: 단일 키워드 CTA ("DM으로 '진단' 보내세요")
- 측정: 주간 DM 유입 수 트래킹

### 병목2: DM → 카카오톡 전환
- 현재: 30~50%
- 목표: 70%
- 핵심 레버: DM 3회 이내 카카오톡 전환
- 측정: DM 문의 중 카카오톡 전환 비율

### 병목3: 카카오톡 → 계약
- 현재: 20~30%
- 목표: 40%
- 핵심 레버: 구조화된 상담 4단계 (진단→공감→제안→클로징)
- 측정: 상담 건 중 계약 전환 비율
